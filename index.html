About the project

This is a visualization of data from Philadelphia's Bike Sharing scheme Indego. The aim is to explore ways of both visualizing exact and aggregated data in a way that will allow viewers to have fun, and also discover the underlying structure of the data for themselves. This project is done in fulfilment of a course requirement at Barcelona Graduate School of Economics.
